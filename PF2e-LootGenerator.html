About This Tool

Acknowledgements

This tool came about in the midst of work crunch while I was learning the system and could barely remember to award loot to my poor players. I hope it saves you hours combing through to find treasure (as it already has for me)! I maintain this tool alongside updates to my personal Pathfinder 2e Obsidian vault so bear with me if updates are slow. Feel free to get in touch if you have feedback or questions.

This tool was designed by me and created with the assistance of AI. This project wouldn't be possible without the work of the following community projects:

  • The absolutely indispensible Archives of Nethys
  • The immense time savings of Obsidian TTRPG Tutorials
  • The wonderful folks maintaining the Foundry VTT PF2e Game System
  • MostTornBrain's excellent PF2E Markdown Exporter
